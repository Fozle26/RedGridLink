Frequently Asked Questions

What is Red Grid Link?

Red Grid Link is an offline-first team tracking and MGRS navigation app for small teams of 2-8 people. It uses encrypted Bluetooth to sync team positions without cell service, servers, or accounts.

How is this different from ATAK?

Red Grid Link is a lightweight ATAK alternative designed for civilian teams. It works on both iOS and Android (ATAK is Android-only), requires no TAK Server, sets up in 30 seconds, and uses Bluetooth instead of server infrastructure. ATAK is built for military operations; Red Grid Link is built for SAR, hunting, and backcountry teams.

Does it work without cell service?

Yes. Red Grid Link is 100% offline-capable. Download topo maps ahead of time, navigate with GPS-based MGRS, and sync team positions over Bluetooth. No internet connection is ever required.

How does Field Link team sync work?

Field Link uses Bluetooth Low Energy and WiFi Direct to automatically discover nearby devices and share encrypted position data. No pairing, no configuration. Turn it on and your team appears on the map. All data is AES-256-GCM encrypted with ephemeral session keys.

What is MGRS?

MGRS (Military Grid Reference System) is the coordinate system used by NATO forces worldwide. It provides precise grid references that are easier to communicate over radio than latitude/longitude. Red Grid Link displays live MGRS coordinates with up to 1-meter precision.

Is my data private?

Completely. Red Grid Link has no accounts, no cloud sync, no analytics, no tracking, and no ads. Location data stays on your device. Field Link data is ephemeral and disappears when the session ends. All communication is AES-256-GCM encrypted.

What's the range of Field Link?

Bluetooth Low Energy range is typically 30-100 meters in open terrain. WiFi Direct extends this to approximately 200 meters. For longer range, mesh relay support is planned for a future version.
