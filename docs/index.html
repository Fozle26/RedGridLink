Frequently Asked Questions

How does Field Link work?

Field Link uses Bluetooth Low Energy and WiFi Direct to auto-discover nearby teammates and share encrypted position data. No pairing, no configuration. Turn it on and your team appears on the map. All communication is AES-256-GCM encrypted with ECDH ephemeral session keys.

Does it work without cell service?

Yes. Red Grid Link uses GPS satellites for positioning and Bluetooth for team sync. No cell towers, no WiFi, no internet required. Download maps ahead of time for full offline operation.

What data do you collect?

None. No accounts, no analytics, no cloud sync, no tracking, no ads. Location data stays on your device. Field Link session data is ephemeral and disappears when the session ends.

How is this different from ATAK?

Red Grid Link is lighter, cross-platform (iOS + Android), requires no TAK Server, and is designed for civilian teams. ATAK is a powerful military tool but requires Android, server infrastructure, and significant setup time. Red Grid Link gets your team on the map in 30 seconds.

Can I use it for hunting?

Yes. Hunting mode adapts the UI with hunting-specific terminology, stand locations, and party tracking. Share positions with your hunting party over Bluetooth without cell service.

When is Android coming?

Red Grid Link is built with Flutter, a cross-platform framework. The Android build is functional and launch is coming soon. Star the GitHub repo to stay updated.
